## 📌 Descripción

Este repositorio contiene una guía completa para el uso de Git y GitHub, desde comandos básicos hasta prácticas profesionales utilizadas en entornos reales de desarrollo.

Incluye:
- Configuración inicial

- Manejo de ramas
- Buenas prácticas
- CI/CD básico

---

## 🚀 ¿Qué es Git?

Git es un sistema de control de versiones que permite:
- Registrar cambios en el código
- Trabajar en equipo
- Gestionar versiones
- Controlar historial

---

## 🌐 ¿Qué es GitHub?

GitHub es una plataforma en la nube que permite:
- Almacenar repositorios
- Colaborar con equipos
- Automatizar procesos (CI/CD)
